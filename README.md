# Marcos Galvão

Quality Assurance focado em **automação de testes Web e API**.

Neste GitHub, compartilho projetos de estudo e prática relacionados à criação, estruturação e evolução de testes automatizados, utilizando principalmente **Playwright**, **TypeScript** e ferramentas voltadas para validação de aplicações Web e APIs.

---

## Sobre este perfil

Este perfil é voltado para projetos de automação de testes, com foco em:

- Automação de testes Web
- Testes automatizados de API
- Estruturação de projetos de testes
- Boas práticas em cenários automatizados
- Validação de fluxos críticos
- Organização de massa de testes
- Reutilização de helpers, fixtures e page objects
- Execução de testes em diferentes ambientes
- Estudos com Playwright, TypeScript e testes de API

## Projetos

Os repositórios deste perfil são direcionados para estudos e práticas de automação de testes, incluindo:

- Projetos com Playwright
- Testes automatizados para aplicações Web
- Validação de fluxos ponta a ponta
- Testes de API
- Estruturação de cenários reutilizáveis
- Organização de fixtures, page objects, helpers e services
- Validação de respostas, contratos e cenários críticos
- Execução local e integração com pipelines

---

## Objetivo

Utilizar este espaço para documentar minha evolução prática em automação de testes, explorando padrões, ferramentas e estratégias que ajudem a construir suítes de testes mais confiáveis, organizadas e fáceis de manter.
